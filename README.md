💈 Barbería Web

Sitio web desarrollado para promocionar servicios de barbería, mostrar trabajos realizados y facilitar el contacto directo con potenciales clientes.

🌐 Sitio Web

(En desarrollo)

📋 Descripción

Este proyecto consiste en una página web desarrollada utilizando HTML y CSS, orientada a la promoción de servicios de barbería.

La web busca brindar una experiencia visual atractiva para los usuarios, permitiendo conocer los servicios ofrecidos, visualizar estilos de corte y comunicarse de forma rápida mediante WhatsApp.

✨ Características

- Diseño responsive adaptable a dispositivos móviles y computadoras.
- Presentación de servicios de barbería.
- Galería de trabajos realizados.
- Información de contacto.
- Integración con WhatsApp.
- Navegación simple e intuitiva.
- Diseño orientado a la captación de clientes.

🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- Git
- GitHub

🎯 Objetivo del Proyecto

Desarrollar un sitio web moderno y funcional para promocionar servicios de barbería, mejorar la presencia digital del negocio y facilitar la comunicación con potenciales clientes.

🚀 Aprendizajes Obtenidos

A través de este proyecto pude fortalecer conocimientos en:

- Estructuración de páginas web con HTML.
- Diseño y estilos mediante CSS.
- Diseño responsive.
- Organización de contenido visual.
- Integración de enlaces externos.
- Gestión de repositorios con Git y GitHub.
- Desarrollo de proyectos web orientados a negocios reales.

🚧 Estado del Proyecto

🔨 Proyecto en desarrollo.

Próximamente se incorporarán mejoras visuales, optimización del diseño y nuevas funcionalidades para mejorar la experiencia de los usuarios.
